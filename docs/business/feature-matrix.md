Use this matrix during sales scoping and renewal conversations.

| Feature Category | Basic | Pro | Premium |
|---|---|---|---|
| Branded app experience (logo/colors/contact) | Included | Included | Included |
| Admin dashboard | Included | Included | Included |
| Defendant portal | Included | Included | Included |
| Live selfie check-ins | Included | Included | Included |

| GPS/location capture for check-ins | Included | Included | Included |
| Reminder workflow tuning | Standard | Advanced rules | Advanced + strategy support |

| PWA install branding | Included | Included | Included |
| Custom domain setup | Optional add-on | Included | Included |
| Priority support | No | Yes | Highest priority |
| Custom integrations | Add-on | Add-on | Scoped roadmap option |
| Custom reporting | Add-on | Limited custom | Expanded custom |
| Quarterly roadmap sessions | No | Optional add-on | Included |

## Notes
- “Included” means within standard scope and monthly support boundaries.
- “Add-on” means separate statement of work and billing.
- Keep tier promises conservative; avoid committing custom engineering inside standard support unless explicitly priced.
