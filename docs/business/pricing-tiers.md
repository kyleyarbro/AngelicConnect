# Pricing Tiers

## Purpose
Use this as an internal baseline for quoting agencies. Final pricing should reflect agency size, risk tolerance, and custom scope.

## Suggested Structure

### Basic
- Setup fee: one-time onboarding, branding, and launch configuration.
- Monthly fee: hosting, monitoring, backups, and standard support.
- Includes: branded app, admin + defendant portals, selfie check-ins, standard reminders, basic reporting.

### Pro
- Setup fee: includes Basic setup plus expanded workflow configuration.
- Monthly fee: higher SLA, enhanced operations support, and additional admin capabilities.
- Includes: everything in Basic plus deeper activity tracking, configurable reminder flows, and priority support queue.

### Premium
- Setup fee: includes discovery + tailored implementation planning.
- Monthly fee: dedicated support lane, advanced reporting, and higher-touch operations support.
- Includes: everything in Pro plus custom domain support, premium escalation options, and quarterly roadmap sessions.

## Billing Components

### Bill separately (recommended)
- New feature development outside current product behavior.
- Third-party integrations (payment tools, CRM, telephony, reporting platforms).
- Data migration/cleanup from legacy systems.
- Brand redesign beyond standard theme/icon package.
- Emergency after-hours support beyond contracted response targets.

### Keep inside monthly fee
- Routine hosting/runtime upkeep.
- Security patching and dependency maintenance.
- Minor UI/content updates within existing components.
- Normal bug triage and fixes.

## Internal Quoting Notes
- Quote setup as fixed-scope whenever possible.
- Quote custom items as scoped project phases with clear acceptance criteria.
- Re-price annually as support load and feature depth increase.
